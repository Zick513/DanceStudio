# DanceStudio
Projekt praktyczny w firmie PixelShark

Opis projektu 
Projekt przedstawia nowoczesną stronę typu onepage dla instruktorki tańca, która prowadzi zajęcia grupowe dla początkujących i średniozaawansowanych uczestników.
Strona została zaprojektowana jako landing page zachęcający użytkowników do zapisania się na zajęcia taneczne. Projekt zawiera informacje o stylach tańca, grafiku zajęć, zaletach szkoły oraz formularz kontaktowy.
Na stronie będą znajdować się takie sekcje jak: Strona główna, o mnie, style taneczne, grafik zajęć, style tańców, formularz kontaktowy.

Czego nauczyłem się pod czas pracy z projektem:
1. Tworzenie responsywnej strony internetowej
2. używania Grid i FlexBox w CSS
3. pracy z sekcjami onepage oraz nawigacją między nimi.
4. Lepiej ogarniam kod HTML i CSS

Z czym  miełem problem?
1. stworzeniem burger menu
2. pracą z responsywnym zdjęciem w sekcji hero
3. dostosowaniem strony do urządzeń mobilnych
